Необходимо реализовать сервис с следующим функционалом с на языке Python 
использованием фреймворка Django.
1. В базе данных Postgres должна быть таблица currency c колонками:
    - id — первичный ключ
    - name — название валюты
    - rate — курс валюты к рублю

2. Должна быть консольная команда для обновления данных в таблице currency. Данные
по курсам валют можно взять отсюда: http://www.cbr.ru/scripts/XML_daily.asp

3. Реализовать 2 REST API(JSON RPC 2.0) метода:
    - GET /currencies — должен возвращать список курсов валют
        -- возможность пагинации
    - GET /currency/ — должен возвращать курс валюты для переданного id

4. API должно быть закрыто bearer авторизацией.

5. Все покрыть юнит тестами

6. Результат выложить на Github и прислать мне ссылку.
